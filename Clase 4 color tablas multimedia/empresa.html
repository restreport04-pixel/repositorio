Curso de Introducción al Desarrollo Web

¿De qué trata el curso?

Este curso esta diseñado para enseñar los fundamentos basicos del desarrollo web utilizando HTML. Aprenderás a crear páginas web estructuradas y listas para publicarsen en internet.

¿A quién va dirijido?

Esta dirigido a estudiantes principiantes que desean iniciar en el mundo de la programación y el desarrollo de sitios web desde cero.

¿Qué aprenderás?

Durante el curso aprendrás a utilizar etiquetas básicas de HTML, insertar imágenes, videos, mapas y construir tablas para organizar información.

Beneficios del curso

  • Aprendizaje desde cero.
  • Ejercicios prácticos.
  • Proyecto final tipo landing page.
  • Certificado de participación.

Pasos para inscribirse

  1. Completar el formulario.
  2. Realizar el pago.
  3. Recibir información por correo.
  4. Comenzar las clases.
